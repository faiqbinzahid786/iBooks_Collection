Delete All Nodes Documentation


Delete All Nodes allows user to delete the complete record of the book they desrired to remove from the system. Below is the picture of Delete All Nodes tab:



Moreover, below is the picture the dropdown from "Please choose an ID below" field:



Form requirements and action:

  1. Please choose an ID below: represent the unique ID of a book. This will allow users to choose the Book ID that is already in the system and delete it. Example: bk101, bk102, etc.

  2. The Delete button will delete the book from the system. After that it will redirect to the View Collection page so that the user can proof check of thier action.
