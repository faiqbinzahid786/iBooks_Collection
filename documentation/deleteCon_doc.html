Conditional Delete Documentation


Conditional Delete will display a form which will allow user to delete perticular information about the book of thier choice. Below is the picture of Conditional Delete tab:



Form requirements and action:

  1. Please choose an ID below: represent the unique ID of a book. This will allow users to choose the Book ID that is already in the system that they want to remove information of. Example: bk101, bk102, etc.

  2. Please choose a tag below: represent the information about the specific book. In our system we have: Author, Title, Genre, Price, Published Date and Description. So the user will choose the desired category of information they want to remove.

  3. The Delete button will delete the information of the perticular book that they user choose in the box fields above. After that it will redirect to the View Collection page so that the user can proof check of thier action.
